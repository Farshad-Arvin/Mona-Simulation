-----------------------------------
CONTROLLERS (C++)
-----------------------------------
beeclustArduino - program used with the real Monas to implement beeclust
beeclustStage - Stage controller used for implementing beeclust

-----------------------------------
STAGE INCLUDE FILES
-----------------------------------
mona.inc - Mona model file
map.inc - used to create walls/obstacles in worlds
gradientAll.inc - used to implement the colour gradients modeling light 
		  intensity

-----------------------------------
STAGE WORLD FILES
-----------------------------------	
exp1_large.world - Single light source, large arena with many robots. Comment 
		   any robots not needed
exp2_large.world - Two light sources of different intensities. Large arena with 
		   many robots
experiment1.world - One light source. Arena the same size as the Lego arena used 
		    for hardware testing.
experiment2.world - Two light sources. Arena the same size as the Lego arena used 
		    for hardware testing.
